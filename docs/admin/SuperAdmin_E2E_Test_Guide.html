6. Test PageSpeed Insights NEW

The PageSpeed Insights feature provides real-time Core Web Vitals monitoring with AI-powered recommendations.

  1. From dashboard, locate the PageSpeed Insights card
  2. Verify: Card shows latest scores or "No data yet"
  3. Click Scan Now button
  4. Verify: Loading indicator appears
  5. Verify: After 15-30 seconds, scores update

Expected Metrics:

Metric Target Description
Performance (Mobile) 85+ Load speed, interactivity on mobile
Performance (Desktop) 95+ Load speed, interactivity on desktop
Accessibility 95+ WCAG compliance, screen reader support
Best Practices 95+ HTTPS, security headers, etc.
SEO 95+ Meta tags, crawlability, mobile-friendly

Automated Weekly Scans:

A cron job runs every Friday at 9 AM UTC to scan PageSpeed and email results to santhosh@finacegroup.com

9. Test Logout & Security

  1. Click Logout button (top right)
  2. Verify: Redirected to login page
  3. Try accessing dashboard URL directly:
    https://finaceverse.io/superadmin-dashboard
  4. Verify: Redirected back to login (protected route)

Security Features to Verify:

  • JWT tokens expire after session
  • Rate limiting on login attempts (10 per 15 min)
  • HTTPS enforced on all endpoints
  • CSP headers block inline scripts
  • Session tokens cleared on logout

10. API Endpoints Reference

Endpoint Method Description
/api/superadmin/login POST Authenticate with master key + password
/api/products GET/POST List or create products
/api/products/:id PUT/DELETE Update or delete product
/api/analytics/pagespeed GET Get latest PageSpeed scores
/api/analytics/pagespeed/scan POST Trigger new PageSpeed scan
/api/cron/pagespeed-weekly POST Weekly cron endpoint (requires X-Cron-Secret)
/api/seo/scan POST Run SEO audit on all pages
