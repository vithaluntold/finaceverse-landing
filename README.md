# FinACEverse Landing Page

A modern React landing page for FinACEverse - The World's First Cognitive Operating System for Finance.

## Features
- 7 Core Modules: VAMN (Brain), Cyloid (Eyes), Accute, Luca, Finaid Hub, Finory & EPI-Q, Sumbuddy
- Responsive design with video backgrounds
- Interactive UI components
- SEO optimized with React Helmet

## Tech Stack
- React 17
- React Router v5
- CRACO for custom build configuration
- Modern CSS with CSS variables

## Development

```bash
npm install
npm start
```

## Production Build

```bash
npm run build
```

## Deployment
This project is configured for Railway deployment with static file serving.
